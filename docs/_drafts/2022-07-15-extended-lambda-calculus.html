As we all know, the function type $A\impl B$ corresponds to implication. We can make the lambda calculus more symmetric by adding a dual function type $A\impr B$.

Functions in types correspond to those types being true. Symmetrically, we should have a dual function in a type if that type is false.

The rules for function applications are derived from generalized modus ponens. By workign through the cases, one finds the following rules:

  • If $a$ is a function of type $A$ and $b$ is a function of type $A\impl B$ then $a b$ ia a function of type $B$.
  • If $a$ is a dual function of type $A$ and $b$ is a dual function of type $A\impr B$ then $ab$ is a dual function of type $B$.

The lambda-constructors are the analogues of the deduction theorem:

  • $\lambda a:A.b$: a function of type $A\impl B$
    "Given $A$ leads to $B$, deduce $A$ implies $B$"
    Suppose $b$, a function, has type $B$ given $a$, a function, has type $A$. Then form $\lambda a:A.b$.
  • $[a,\overline b]$: a function of type $A\impr B$
    "Given $A$ is true and $B$ is false, deduce $A\impr B$"
    This can be formed given any function and dual function.
  • $[\overline a,b]$: a dual function of type $A\impl B$
    "Given $A$ is false and $B$ is true, deduce $A\impl B$ is false"
    This can be formed given any function and dual function.
  • $\overline\lambda a:A.b$: a dual function of type $A\impr B$
    dual of "Given $A$ leads to $B$, deduce $A$ implies $B$"
    Suppose $b$, a dual function, has type $B$ given $a$, another dual function, has type $A$. Then form $\overline\lambda a:A.b$.

Reduction rules: the outcome of modus ponens

  • Let $\lambda a:A.b$ be a function and $c$ be a function of type $A$. Then $(\lambda a:A.b) c=b[a/c]$.
  • Dually, let $\overline\lambda a:A.b$ be a dual function and $c$ bea dual function of type $A$. Then $(\overline\lambda a:A.b) c=b[a/c]$.
